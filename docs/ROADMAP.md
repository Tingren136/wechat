# Roadmap

## Phase 1: 项目说明固化

- 补齐 README
- 写清楚现状、目标、输入输出
- 让其他 AI 可以无上下文接手

## Phase 2: 预览原型产品化

- 清理预览脚本
- 减少对绝对路径的依赖
- 固化文章输入与输出目录
- 保证 4 套 HTML 稳定生成

## Phase 3: 工作流拆段

- 正文润色阶段
- Markdown 结构化阶段
- 配图规划与提示词打包阶段
- 正文图片生成阶段
- 4 套 HTML 输出阶段

## Phase 4: Skill 封装

- 统一 skill 输入参数
- 明确每个确认点
- 输出中间产物与最终产物
- 准备示例与文档

## Phase 5: GitHub 发布

- 新建独立仓库
- 整理项目目录
- 加入示例素材与截图
- 补齐安装与使用说明
